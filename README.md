# Cytoid Info Querier(Compose)

一个安卓端的提供[Cytoid](https://cytoid.io/)游戏查询B30（不只30）和R10（当然也不只10）的工具，并不止步于此，更多功能正在绝赞贵阳中

## 运行平台

Android 7.0+

## 分支策略

-  ```main```：用于发布版本更新和重要内容更改
-  ```dev``` ：用于日常开发
